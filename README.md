# Network-Scanner

Scanner de rede local utilitário baseado no protocolo ARP utilizando a biblioteca `Scapy` para deteção de endereços IP e endereços MAC ativos.

# Características
- Envio e receção de tráfego de rede nativo através de Pacotes Ethernet / ARP.
- Identificação rápida de dispositivos ligados à mesma rede local.

# Como Executar
1. Instale as dependências requeridas:
   
   pip install -r requirements.txt
